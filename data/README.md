# Data Directory

## Structure

```
data/
├── raw/           # Raw CSV files (read-only, version-controlled)
│   ├── jay_chou_master_dataset.csv   — Full merged dataset (all features)
│   ├── jay_discography.csv           — Album/track metadata
│   ├── spotify_features.csv          — Spotify audio features only
│   ├── lyrics.csv                    — Song lyrics text
│   ├── album_covers.csv              — Album cover URLs
│   └── song_tags.csv                 — Style tags in long format
│
└── processed/     # Preprocessed analysis outputs (gitignored, regenerable)
    ├── songs.json               — Unified song data for frontend
    ├── albums.json              — Album-level aggregate data
    └── jay_music_dataset.csv    — Tabular dataset (26 columns, 174 rows)
```

Raw data was sourced from Spotify Web API and manual collection.
Processed files are generated by `src/analysis/preprocess.py`.
