# Frontend Data Files

This directory contains JSON data consumed by the frontend SPA at runtime.

| File | Type | Source |
|------|------|--------|
| `songs.json` | Symlink | `data/processed/songs.json` |
| `albums.json` | Symlink | `data/processed/albums.json` |
| `recommender_data.json` | Static | Generated by `src/analysis/song_recommender.py` |

The `songs.json` and `albums.json` files are **symlinks** to `data/processed/`, 
so after running `make data` the frontend always reads the latest data.

**To regenerate all data:**
```bash
make data                          # Preprocessing → updates JSON in data/processed/
python3 -m src.analysis.song_recommender  # Regenerates recommender_data.json
```

**To start the frontend dev server:**
```bash
cd src/frontend
npm install    # One-time setup
npx vite dev   # Dev server on port 5173
```
